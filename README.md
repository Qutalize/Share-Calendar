# CalSync

**CalSync** は、フレンド間でスケジュールを共有できる拡張型カレンダーアプリです。  
予定の重複検知・フレンド申請・承認済みユーザー間の予定共有など、チームや友人間のスケジュール調整に役立つ機能を備えています。

## 主な機能

### ユーザー登録・認証（`auth.py`）
- 新規アカウントの作成とログイン / ログアウト
- ユーザーごとに独立したカレンダーを管理
- 
### カレンダー・予定管理（`event_service.py`）
- カレンダー上への予定の追加・編集・削除
- 予定のステータス管理（**確定** / **仮**）
- **⚠️ ダブルブッキング警告**：既存の予定と時間が重複する場合、登録前に警告を表示

### フレンド機能（`friend_service.py`）
- ユーザー名で他のユーザーを検索してフレンド申請を送信
- 受信した申請を **承認 / 拒否** で管理
- フレンド一覧の表示・削除

### 通知機能（`notification_service.py`）
- フレンド申請の受信・承認などのイベントをユーザーに通知

### 予定の共有
- フレンド承認後、お互いの **「確定」済み予定** が相手のカレンダー上に表示される
- 未確定（仮）の予定は共有されない

## 通信構成

本アプリは **VS Code Dev Tunnel** を使用してローカル開発環境を外部に公開しています。

```
[ブラウザ / スマートフォン]
        │
        │ HTTPS
        ▼
[VS Code Dev Tunnel]
(vscode.dev/tunnel/...)
        │
        │ localhost
        ▼
[Flask 開発サーバー（main.py）]
```
- `localhost` に限らず、外部ネットワークからもアクセス可能
- Dev Tunnel により HTTPS が自動付与されるため、安全な通信を確保
- チームメンバーやテスターが URL を共有するだけで動作確認できる

## 使い方

1. **アカウント作成**：トップページからサインアップ
2. **予定を登録**：カレンダー上の日付をクリックして予定を追加
   - 既存の予定と時間が重複する場合は⚠️警告が表示されます
3. **フレンド申請**：ユーザー検索から相手を見つけて申請を送信
4. **申請を承認**：通知から申請を確認し、承認 / 拒否を選択
5. **予定を共有**：承認後、お互いの確定済み予定がカレンダーに表示されます
